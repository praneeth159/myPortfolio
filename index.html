Projects

Music Recommendation system

This is a music recommendation project designed with help of deep learning techniques which aims to provide users with personalized music recommendations tailored to their musical taste.

I collected vast amounts of music data from various sources. The data encompass diverse genres, artists, albums. To make the data suitable for deep learning models, extensive preprocessing is performed. Techniques like feature extraction, audio analysis, and metadata enrichment are applied. Deep Learning Models like Convolutional Neural Networks are used to learn intricate patterns and dependencies in music data. The application creates individual user profiles by analyzing their listening history, favorite genres, liked artists, by understanding each user's unique taste, the system can tailor recommendations that align with their preferences. In addition to individual profiling, the project incorporates collaborative filtering techniques to identify users with similar tastes. This allows the application to recommend songs loved by users with comparable musical preferences

toilet finder-PottySpoty

PottySpoty is an ingenious toilet finder application developed using Java and mobile computing technologies. Aimed at providing users with a hassle-free solution to locate nearby public restrooms.

PottySpoty harnesses the robust Google Maps API to access precise geolocation data. By utilizing GPS and mapping functionalities, the app accurately identifies the user's current location and displays nearby public toilets on an interactive map.Upon launching the app, users can quickly view nearby toilets on the map, access essential information such as restroom availability, amenities, and user ratings/reviews.With Google Maps integration, PottySpoty allows users to seamlessly navigate to the selected restroom with just a tap on the map. This integration streamlines the process of reaching the chosen destination with ease.Developed using Java and mobile computing principles, PottySpoty is designed to be cross-platform compatible. It is accessible on both Android and iOS devices, ensuring a broader user base and maximum convenience.

Social Distance Detector

This is a social distancing project designed to promote public safety and reduce the spread of contagious diseases in crowded spaces. YOLOv3 pre-trained weights are used for real-time object detection and localization, this application utilizes Python for backend processing and HTML for user interface. The project aims to provide a comprehensive solution to monitor social distancing compliance and alert users when safe distances are not maintained.

This application with the help of YOLOv3 (You Only Look Once version 3) deep learning algorithm accurately detects and locates individuals in crowded areas. The pre-trained weights enable real-time and efficient processing of live video streams, ensuring swift detection of people within the monitored environment. Once individuals are detected, it analyzes their positions in real-time to determine whether safe social distancing guidelines are being followed. The system measures the distances between individuals and identifies potential violations.When the system detects instances of inadequate social distancing, it triggers real-time alerts to notify users and relevant authorities. Alerts can be in the form of visual indicators on the user interface, sound notifications. It offers an interactive user-interface developed using HTML. Users can easily access the system through their web browsers, view live video streams, and monitor social distancing compliance in real-time.

Video converter and image editor

This is a video and image editor project developed using Python. Combining the power of Python libraries for image processing and video manipulation, it provides users with a user-friendly and straightforward platform to edit their visual content.

PyEditor allows users to perform fundamental image editing tasks, such as cropping, resizing, and adjusting brightness, contrast.After completing the edits, users can export their edited images and videos and save them for sharing or further use. The project is designed to be lightweight and efficient, ensuring quick processing and minimal resource consumption on the user's computer.
